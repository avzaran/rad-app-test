# Frontend

Figma-based UI implementation (design contract locked).

## Commands

```bash
npm i
npm run dev
npm run build
npm run typecheck
npm run test:unit
npm run test:visual
npm run design:check
```

## Environment

Use `.env.example`:

- `VITE_API_BASE_URL` - backend URL
- `VITE_USE_MOCK_API` - `true` in dev for local mock fallback
